The problem

LLMs are brilliant. Bodies are dumb. There's no bridge.

An LLM enters an unknown machine the same way a human enters an unfamiliar SQL terminal — blind. AISocket gives every device a readable identity: what it can do, what it must never do, and how to ask for help when it gets stuck.

How it works

Four steps. Any device.

01 — passport

Body declares itself

Capabilities, forbidden actions, risk class, autonomous mandate — all in one structured object.

02 — law

Every command validated

6-layer check: firmware rules → capabilities → emergency → offline mandate → delegation → context.

03 — connect

LLM wears the body

The LLM receives a structured prompt — not raw UI. It speaks commands, not clicks.

04 — learn

Community improves it

Bugs, tactics, benchmarks — all go into the shared prompt repository. Every body learns from every incident.

Code

Five lines to get started.

$ pip install aisocket
# define your body's passport
passport = Passport("mower_v1")
passport.capability("mow_grass", "report_status")
passport.forbidden("harm_human", "enter_building")
passport.connect(llm="gemini-flash")

# LLM now operates within body law
